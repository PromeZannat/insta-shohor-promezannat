2. Difference between Local Storage & Session storage?

The main difference between local storage & session storage is that data in the local storage doesn't deleted after the page or browser closed. But data in session storage automatically deleted or cleard after the page or bowser closed. In local storage, if any one wants to clear data the have to do it manually. But for session storage, it has expires based setting. In session storage, one can write the data in the browsers memory. But in local storage, one can write the data in disk.
